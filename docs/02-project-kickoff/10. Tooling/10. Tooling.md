### 2. Stack Oficial de Trabajo (Desarrollo y Gestión)
| Categoría | Herramienta Elegida | Justificación |
| --- | --- | --- |
| IDE Backend | IntelliJ IDEA / Eclipse | Entornos oficiales y estándar de la industria para desarrollo en Java. Excelente debugging. |
| IDE Frontend | VS Code | Ligero, soporte absoluto para JavaScript y extensiones de Vite/ESLint. |
| Cliente de Base de Datos | pgAdmin / DBeaver | Necesario para inspeccionar y alterar esquemas en PostgreSQL local y remoto. |
| Gestión de Proyecto | Jira / Trello | Para llevar el backlog formalizado bajo metodologías Agile y trazar el progreso de desarrollo. |
| Diseño UI | Figma | Única fuente de verdad visual para el diseño de interfaces web en JS. |
| Control de Versiones | Git + GitHub | Repositorio central del código fuente y pipelines de integración (CI/CD). |

### 3. Entorno de Ejecución Local
Todos los desarrolladores deben tener instalado:
- **Java JDK (17 o superior):** Para compilar el backend.
- **Node.js y npm/yarn:** Para empaquetar y levantar el servidor de desarrollo del Frontend (Vite).
- **Docker (Opcional pero recomendado):** Para levantar un contenedor de PostgreSQL idéntico al de producción sin ensuciar la máquina local.
