# Inventario Maestro de Paginas (All Pages Index)

**Project:** Nos Fuimos de Finca
**Phase:** 4 System Modeling (D3)

Este documento es el **Mapa Cartografico del Proyecto Frontend**. Enlista todas las pantallas fisicas (Rutas URL) que existiran en la aplicacion y declara que Modulos Funcionales (`MOD-X`) le dan vida a cada una. 

La IA (Director UX) usara este indice para generar los Blueprints individuales en la carpeta `pages/`. El Disenador Humano usara este indice para saber cuantas mesas de trabajo debe crear en Figma.

---

## Mapa Topologico (Sitemap Visual)

Este diagrama muestra la estructura de navegacion y como las rutas anidadas (ej. el Dashboard) comparten un mismo Layout envolvente.

```mermaid
flowchart TD
    %% Raiz de la Aplicacion
    Root((Dominio Principal))
    
    %% Ecosistema B2C
    subgraph B2C [Rutas Publicas B2C - Marketplace]
        direction TB
        Home[Home Page<br>Ruta: /]
        Search[Buscador<br>Ruta: /search]
        Finca[Perfil Finca<br>Ruta: /finca/slug]
        
        subgraph Embudo [Embudo de Pago Isolado]
            Checkout[Checkout<br>Ruta: /checkout/id]
            Success[Recibo<br>Ruta: /checkout/success]
            Checkout --> Success
        end
        
        Home --> Search
        Home --> Finca
        Search --> Finca
        Finca --> Embudo
    end
    
    %% Ecosistema B2B
    subgraph B2B [Rutas Privadas B2B - Portal Finqueros]
        direction TB
        HostLanding[Ventas Aliados<br>Ruta: /host]
        Login[Acceso<br>Ruta: /login]
        Registro[Registro<br>Ruta: /registro]
        Onboarding[Muro KYC<br>Ruta: /onboarding/*]
        
        subgraph DashboardLayout [Layout Padre: Dashboard Sidebar]
            direction LR
            DashHome[Finanzas<br>Ruta: /dashboard]
            DashFincas[Fincas<br>Ruta: /dashboard/fincas]
            DashCal[Calendario<br>Ruta: /dashboard/calendario]
            DashRes[Reservas<br>Ruta: /dashboard/reservas]
        end
        
        HostLanding --> Login
        Login --> Registro
        Registro --> Onboarding
        Login --> DashboardLayout
        Onboarding --> DashboardLayout
    end
    
    Root --> B2C
    Root --> B2B
    
    %% Manejador Global
    Error[Manejo Errores<br>Ruta: Dinamica 404/500]
    Root -.-> Error
```
