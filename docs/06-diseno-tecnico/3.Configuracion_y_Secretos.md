## 2. Variables de Entorno Backend (Spring Boot)

Este es el contrato estricto del archivo `.env.example` que requiere el servidor backend. Las contraseñas reales se inyectarán vía AWS Systems Manager Parameter Store.

```env
# ==========================================
# BACKEND SECRETS (.env.example)
# ==========================================

# 1. CORE APPLICATION
PORT=8080
SPRING_PROFILES_ACTIVE=dev

# 2. DATABASE (PostgreSQL RDS)
# Nunca quemar la contraseña de DB en código.
DB_HOST=localhost
DB_PORT=5432
DB_NAME=nosfuimos_db
DB_USERNAME=postgres
DB_PASSWORD=your_db_password_here

# 3. SECURITY (JWT & Bcrypt)
# Secret simétrico para firmar los JWT. Debe tener mínimo 256 bits.
JWT_SECRET_KEY=generate_a_strong_random_secret_string
JWT_EXPIRATION_MS=86400000

# 4. EXTERNAL INTEGRATIONS (Wompi - MOD-PAY)
WOMPI_PUBLIC_KEY=pub_test_XXXXXX
WOMPI_PRIVATE_KEY=prv_test_XXXXXX
# Llave secreta para validar la firma de los Webhooks (Integridad)
WOMPI_EVENTS_SECRET=events_test_XXXXXX

# 5. NOTIFICATIONS (SendGrid - MOD-NOT)
SENDGRID_API_KEY=SG.xxxxxxxxxxxxxx
DEFAULT_SENDER_EMAIL=reservas@nosfuimosdefinca.com

# 6. AWS STORAGE (Amazon S3 - MOD-FCNT)
AWS_ACCESS_KEY_ID=AKIAXXXXXXX
AWS_SECRET_ACCESS_KEY=xxxxxxxxxxxxxxxx
AWS_REGION=us-east-1
S3_BUCKET_NAME=nosfuimos-fincas-assets
```

## 3. Variables de Entorno Frontend (Next.js)

Este es el contrato para el cliente de React. Todas las variables que se requieran del lado del cliente en el navegador deben estar precedidas por `NEXT_PUBLIC_`.

```env
# ==========================================
# FRONTEND SECRETS (.env.example)
# ==========================================

# URL Base del backend Spring Boot. 
# En dev: http://localhost:8080. En prod: https://api.nosfuimosdefinca.com
NEXT_PUBLIC_API_URL=http://localhost:8080

# Llave PÚBLICA de Wompi para cargar el Widget de pagos en el navegador.
# ES SEGURO exponer esta llave en el frontend.
NEXT_PUBLIC_WOMPI_KEY=pub_test_XXXXXX

# Dominio del CDN de AWS Cloudfront (Para precargar optimización de imágenes en Next.js)
NEXT_PUBLIC_CDN_DOMAIN=https://d123456.cloudfront.net
```
