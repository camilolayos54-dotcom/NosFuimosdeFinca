# Entregable 7 (D7): Contratos API (OpenAPI)

**Proyecto:** Nos Fuimos de Finca
**Fase:** 6 — Diseño Técnico
**Módulo:** MOD-PAY
**Estado:** Aprobado

## 2. Especificación OpenAPI YAML

```yaml
openapi: 3.0.3
info:
  title: Nos Fuimos de Finca - Payment Webhooks
  version: 1.0.0
paths:
  /api/v1/payments/webhook:
    post:
      summary: Recibir evento transaccional de Wompi
      tags: [Payments]
      description: Endpoint público consumido por Wompi. La validación de seguridad se hace vía firmas (Checksums).
      requestBody:
        required: true
        content:
          application/json:
            schema:
              $ref: '#/components/schemas/WompiWebhookPayload'
      responses:
        '200':
          description: Evento recibido y encolado exitosamente. (Wompi requiere 200 OK rápido)
        '400':
          description: Firma inválida (Bad Request)
components:
  schemas:
    WompiWebhookPayload:
      type: object
      properties:
        event:
          type: string
          example: "transaction.updated"
        data:
          type: object
          properties:
            transaction:
              type: object
              properties:
                id:
                  type: string
                status:
                  type: string
                  example: "APPROVED"
                amount_in_cents:
                  type: integer
                reference:
                  type: string
                  description: "El UUID del Booking"
        signature:
          type: object
          properties:
            checksum:
              type: string
```
