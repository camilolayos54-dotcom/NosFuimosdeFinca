## 2. Diagrama de Clases UML

```mermaid
classDiagram
    class Booking {
        -UUID id
        -UUID fincaId
        -UUID turistaId
        -Date checkIn
        -Date checkOut
        -BigDecimal totalPrice
        -BookingStatus status
        -Date lockedUntil
        -Date createdAt
        +softLock(ttlMinutes: int) void
        +hardLock() void
        +cancel() void
        +isExpired() boolean
    }
    
    class BookingStatus {
        <<enumeration>>
        SOFT_LOCKED
        HARD_LOCKED
        CANCELLED
    }
    
    class BookingService {
        -BookingRepository bookingRepo
        -FincaClient fincaClient
        +createSoftLock(fincaId: UUID, userId: UUID, checkIn: Date, checkOut: Date) BookingDTO
        +confirmHardLock(bookingId: UUID) void
        +cancelExpiredLocks() int
    }
    
    Booking *-- BookingStatus
    BookingService --> Booking
```
