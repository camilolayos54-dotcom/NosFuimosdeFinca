### 3. Procedimiento de Arranque Local

|Paso|Acción|Comando|Salida Esperada|
|---|---|---|---|
|1|Install prerequisites|Install Java, Node, Docker|CLI available|
|2|Clone repository|`git clone <repo_url>`|Folder created|
|3|Configure environment|Copy `.env.example` to `.env`|Environment variables set|
|4|Install dependencies|`cd frontend && npm install`, `cd backend && mvn clean install`|Deps downloaded|
|5|Start DB|`docker-compose up -d db`|PostgreSQL running on 5432|
|6|Start services|`npm run dev` (FE), `mvn spring-boot:run` (BE)|Apps listening on ports 3000 and 8080|
