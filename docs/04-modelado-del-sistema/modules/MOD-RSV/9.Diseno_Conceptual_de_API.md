# Entregable 9 (D9): Diseño Conceptual de API (MOD-RSV)

**Proyecto:** Nos Fuimos de Finca
**Fase:** 4 — Modelado del Sistema
**Módulo:** MOD-RSV (Gestión de Reservas)
**Estado:** Aprobado

### 1. Endpoints API

| Método | Endpoint | Descripción | Autorización Requerida | Payload de Petición | Respuesta Exitosa |
| --- | --- | --- | --- | --- | --- |
| `POST` | `/api/v1/reservas/soft-lock` | Intenta crear un bloqueo temporal para pago. | `turista-guest` | `{ fincaId: UUID, fechaInicio: YYYY-MM-DD, fechaFin: YYYY-MM-DD }` | `201 Created` `{ reservaId, montoAnticipo, expiresAt }` |
| `GET` | `/api/v1/fincas/:id/disponibilidad` | Consulta el vector de fechas ocupadas para el calendario. | `Público` | `?month=YYYY-MM` | `200 OK` `{ bloqueadas: ["2026-11-06", "2026-11-07"] }` |
| `GET` | `/api/v1/reservas/me` | Lista el historial de reservas del turista actual. | `turista-guest` | None | `200 OK` `[{ id, finca, estado, fechas }]` |
| `GET` | `/api/v1/reservas/propietario` | Lista reservas recibidas en todas las fincas del dueño. | `propietario-admin`| None | `200 OK` `[{ id, turista_nombre, monto, estado }]` |
| `POST` | `/api/v1/reservas/bloqueo-manual` | Propietario bloquea fechas (Mantenimiento). | `propietario-admin`| `{ fincaId: UUID, fechaInicio, fechaFin, motivo }` | `201 Created` |

### 2. Cargas de Error (Excepciones Esperadas)

- `409 Conflict`: 
  ```json
  {
    "error": "CONCURRENCY_ERROR",
    "message": "Las fechas seleccionadas acaban de ser tomadas por otro usuario.",
    "suggestion": "Por favor selecciona un rango diferente."
  }
  ```
- `400 Bad Request`:
  ```json
  {
    "error": "INVALID_DATE_RANGE",
    "message": "La fecha de inicio debe ser anterior a la fecha de fin y superior a hoy."
  }
  ```
