# Entregable 9 (D9): Diseño Conceptual de API (MOD-AUTH)

**Proyecto:** Nos Fuimos de Finca
**Fase:** 4 — Modelado del Sistema
**Módulo:** MOD-AUTH (Autenticación)
**Estado:** Aprobado

### 1. Endpoints API

| Método | Endpoint | Descripción | Autorización | Respuesta Exitosa |
| --- | --- | --- | --- | --- |
| `POST` | `/api/v1/auth/register` | Crea usuario turista. | `Público` | `201 Created` `{ token }` |
| `POST` | `/api/v1/auth/login` | Inicia sesión. | `Público` | `200 OK` `{ token }` |
| `POST` | `/api/v1/auth/forgot-password`| Solicita enlace de reinicio. | `Público` | `200 OK` |
| `GET`  | `/api/v1/auth/me` | Verifica validez del token. | `Cualquiera`| `200 OK` `{ id, nombre, rol }` |
