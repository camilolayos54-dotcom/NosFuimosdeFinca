# Entregable 2 (D2): Flujos de Usuario y Tareas (MOD-AUTH)

**Proyecto:** Nos Fuimos de Finca
**Fase:** 4 — Modelado del Sistema
**Módulo:** MOD-AUTH (Autenticación)
**Estado:** Aprobado

### 1. Flujo de Tarea: Registro de Turista
1. **[Inicio]** Turista intenta reservar sin sesión.
2. Sistema redirige a `/login`.
3. Turista selecciona "Crear Cuenta".
4. Ingresa Email, Nombre y Password.
5. Sistema valida formato y unicidad de email.
6. **[Fin]** Cuenta creada. Auto-login (JWT emitido) y redirección al Checkout pendiente.

### 2. Flujo de Tarea: Recuperación de Contraseña
1. **[Inicio]** Usuario olvida contraseña.
2. Ingresa su email en `/forgot-password`.
3. Sistema valida existencia (sin revelar si existe por seguridad).
4. **[Fin]** Se envía correo con token de 1 hora.
