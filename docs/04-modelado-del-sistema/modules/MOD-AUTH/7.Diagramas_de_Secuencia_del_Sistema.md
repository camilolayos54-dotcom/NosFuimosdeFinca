### 1. SSD: Login y Emisión de JWT

```mermaid
sequenceDiagram
    autonumber
    actor User
    participant Web
    participant API
    participant DB

    User->>Web: Ingresa credenciales
    Web->>API: POST /api/v1/auth/login {email, password}
    API->>DB: Busca usuario por email
    DB-->>API: (hash de password, rol, estado)
    
    alt Usuario suspendido
        API-->>Web: 403 Forbidden
    else Credenciales válidas
        API->>API: Valida bcrypt(password)
        API->>API: Genera JWT (HMAC/RS256) con claims (rol, userId)
        API-->>Web: 200 OK { token }
        Web->>Web: Almacena token en HttpOnly Cookie o LocalStorage
    end
```
