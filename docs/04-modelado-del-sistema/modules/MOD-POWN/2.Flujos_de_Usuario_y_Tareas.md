# Entregable 2 (D2): Flujos de Usuario y Tareas (MOD-POWN)

**Proyecto:** Nos Fuimos de Finca
**Fase:** 4 — Modelado del Sistema
**Módulo:** MOD-POWN (Panel del Propietario)
**Estado:** Aprobado

### 1. Flujo de Tarea: Vista de Dashboard
1. **[Inicio]** Propietario inicia sesión.
2. Es redirigido a `/portal-propietario`.
3. El sistema carga métricas globales (Ingresos del mes, Reservas entrantes).
4. El sistema muestra alertas de acciones requeridas (ej. Responder reseñas).
5. **[Fin]** Propietario navega a sección específica.

### 2. Flujo de Tarea: Gestión de Reservas Recibidas
1. **[Inicio]** Propietario ingresa a `/portal-propietario/reservas`.
2. Visualiza lista de reservas en sus fincas.
3. Puede filtrar por estado (`Pendiente`, `Hard-Lock`, `Cancelada`).
4. Selecciona una reserva para ver detalle (Monto, Fechas, Datos de Contacto del Turista).
5. **[Fin]** Clictea para contactar al Turista vía WhatsApp/Teléfono.
