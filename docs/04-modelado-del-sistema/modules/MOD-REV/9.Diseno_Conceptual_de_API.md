### 1. Endpoints API

| Método | Endpoint | Descripción | Autorización | Respuesta Exitosa |
| --- | --- | --- | --- | --- |
| `POST` | `/api/v1/resenas` | Crea una reseña post-viaje. | `turista-guest` | `201 Created` |
| `POST` | `/api/v1/resenas/:id/reply` | Propietario responde a reseña. | `propietario-admin` | `201 Created` |
| `GET`  | `/api/v1/fincas/:id/resenas` | Lista reseñas de una finca. | `Público` | `200 OK` |
