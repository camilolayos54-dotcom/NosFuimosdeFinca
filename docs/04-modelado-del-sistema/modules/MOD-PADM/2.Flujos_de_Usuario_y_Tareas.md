# Entregable 2 (D2): Flujos de Usuario y Tareas (MOD-PADM)

**Proyecto:** Nos Fuimos de Finca
**Fase:** 4 — Modelado del Sistema
**Módulo:** MOD-PADM (Panel del Administrador)
**Estado:** Aprobado

### 1. Flujo de Tarea: Moderación de Finca
1. **[Inicio]** Admin ingresa a `/admin-panel/fincas`.
2. Visualiza listado de fincas en estado `Pendiente`.
3. Revisa fotos, descripción y documentos del propietario.
4. **[Decisión]** ¿Cumple políticas?
   - SÍ: Clic en "Aprobar". Finca pasa a pública.
   - NO: Clic en "Rechazar/Suspender". Finca se oculta.
5. **[Fin]** Sistema notifica al Propietario.

### 2. Flujo de Tarea: Cancelación por Fuerza Mayor
1. **[Inicio]** Admin es contactado (Soporte) por un problema grave.
2. Ingresa a `/admin-panel/reservas`.
3. Localiza reserva específica mediante ID o nombre.
4. Ejecuta "Cancelar Reserva (Admin Override)".
5. Ingresa motivo de auditoría.
6. **[Fin]** Reserva cancelada y sistema notifica a Turista y Propietario.
