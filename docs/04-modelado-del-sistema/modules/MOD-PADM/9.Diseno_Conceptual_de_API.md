### 1. Endpoints API

| Método | Endpoint | Descripción | Autorización | Respuesta Exitosa |
| --- | --- | --- | --- | --- |
| `GET` | `/api/v1/admin/fincas` | Lista TODAS las fincas (sin filtro de estado). | `sys-admin` | `200 OK` |
| `PATCH`| `/api/v1/admin/fincas/:id/status`| Mutar estado (Aprobar/Suspender). | `sys-admin` | `200 OK` |
| `PATCH`| `/api/v1/admin/reservas/:id/cancel`| Fuerza cancelación y purga Bloqueo Permanente. | `sys-admin` | `200 OK` |
| `PATCH`| `/api/v1/admin/usuarios/:id/ban`| Suspende acceso a usuario. | `sys-admin` | `200 OK` |
