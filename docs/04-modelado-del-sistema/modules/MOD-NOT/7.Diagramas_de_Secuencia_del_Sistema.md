### 1. SSD: Consumo de Eventos y Envío de Correo

```mermaid
sequenceDiagram
    autonumber
    participant EventQueue (RabbitMQ)
    participant Worker (MOD-NOT)
    participant SendGrid API

    EventQueue->>Worker: Consume Event (RESERVA_HARD_LOCK, reservaId)
    Worker->>Worker: Renderiza plantilla HTML con variables
    Worker->>SendGrid API: POST /v3/mail/send
    
    alt Envío Exitoso
        SendGrid API-->>Worker: 202 Accepted
        Worker->>EventQueue: ACK (Mensaje procesado)
    else Falla de API
        SendGrid API-->>Worker: 500 Internal Error
        Worker->>EventQueue: NACK (Reencolar para reintento posterior)
    end
```
