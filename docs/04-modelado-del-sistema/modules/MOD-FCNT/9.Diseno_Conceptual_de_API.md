### 1. Endpoints API

| Método | Endpoint | Descripción | Autorización | Respuesta Exitosa |
| --- | --- | --- | --- | --- |
| `POST` | `/api/v1/fincas` | Crea nueva finca. | `propietario-admin` | `201 Created` |
| `PUT`  | `/api/v1/fincas/:id` | Edita finca existente (RLS). | `propietario-admin` | `200 OK` |
| `POST` | `/api/v1/fincas/upload` | Endpoint multipart para fotos. | `propietario-admin` | `200 OK` `{ url }` |
