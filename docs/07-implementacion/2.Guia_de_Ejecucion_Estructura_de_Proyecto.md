**Proyecto:** Nos Fuimos de Finca
**Fase:** 7 — Implementación
**Estado:** Aprobado

## Contexto de Ejecución
El objetivo de este script es inicializar el entorno técnico. Dado que en la máquina local no se detectó `npm`/`npx`, el frontend se dejará preparado a nivel de carpeta, pero requerirá la ejecución humana del comando de Next.js una vez se instale Node.js.

El Backend (Spring Boot) se inicializa vía el API de Spring Initializr.
La Base de Datos (PostgreSQL) se inicializa vía `docker-compose.yml`.

## Comandos para ejecutar manualmente (Frontend)
1. Instala Node.js (v18 o superior).
2. Abre tu terminal en la raíz del proyecto.
3. Ejecuta:
```bash
npx create-next-app@latest frontend-app --typescript --tailwind --eslint --app --src-dir --import-alias "@/*" --use-npm
```
