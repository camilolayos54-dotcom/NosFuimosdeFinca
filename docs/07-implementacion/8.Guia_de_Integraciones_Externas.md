# Entregable 8 (D8): Guía de Integraciones Externas y Webhooks

**Proyecto:** Nos Fuimos de Finca
**Fase:** 7 — Implementación
**Estado:** Aprobado

## Propósito
Guía para implementar la comunicación con terceros (S3 para imágenes, SendGrid para notificaciones, Wompi para pagos).

## Pasos de Ejecución para el Desarrollador
1. **Wompi Webhook:** Ve al controlador `PaymentController` en `MOD-PAY`. Crea el endpoint `POST /api/payments/webhook`. Extrae el cuerpo crudo y compáralo con el HMAC SHA-256 generado localmente usando `WOMPI_EVENTS_SECRET` (Ver P6-D4 y P6-D8). Solo si la firma coincide, actualiza la base de datos a `APPROVED`.
2. **AWS S3:** En `MOD-FCNT`, implementa un servicio usando el AWS SDK para Java. Crea el método `uploadImage(MultipartFile file)` que retorne la URL pública del CDN.
3. **SendGrid:** En `MOD-NOT`, implementa un `@EventListener` que escuche el evento de dominio `PaymentConfirmedEvent`. Envíale el Payload HTML a la API de SendGrid para el envío de Vouchers.
