# Entregable 7 (D7): Guía de Implementación de Seguridad y Middleware

**Proyecto:** Nos Fuimos de Finca
**Fase:** 7 — Implementación
**Estado:** Aprobado

## Propósito
Guía para proteger el Backend usando Spring Security y filtros JWT sin estado.

## Pasos de Ejecución para el Desarrollador
1. Abre `P6-D4 (Security Implementation)`.
2. En `backend-monolith/src/main/java/com/finca/backend_monolith/core/security/`, crea la clase `JwtAuthenticationFilter`.
3. Implementa la lógica descrita en el seudocódigo: extrae el header `Authorization`, decodifica el token usando la llave simétrica inyectada desde `@Value("${jwt.secret.key}")`, y configura el `SecurityContextHolder`.
4. En `SecurityConfig.java`, configura el `SecurityFilterChain`:
   - Desactiva CSRF (es API REST).
   - Pon la sesión en `STATELESS`.
   - Añade el filtro JWT antes de `UsernamePasswordAuthenticationFilter`.
5. Usa las anotaciones `@PreAuthorize("hasRole('PROPIETARIO')")` en los endpoints especificados.
